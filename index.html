3. Proyectos Realizados

Proyecto 1 – Reciclex (2023)

Desarrollo de una plataforma web para la empresa Reciclex, cuyo objetivo era ubicar puntos de recolección de material reciclable. Nuestra empresa se encargaba de recolectar y vender este material a plantas fabricantes que trabajaban con insumos reciclados. El proyecto fue desarrollado utilizando HTML, CSS, JavaScript y MySQL.

Proyecto 2 – Aplicación con Xamarin (2023)

Elaboración de una aplicación móvil en Xamarin conectada a una base de datos en MySQL. Su funcionalidad principal era la gestión de usuarios mediante un login con validación en tiempo real.

Proyecto 3 – Regulador de Voltaje con Arduino (2024)

Desarrollo de un dispositivo basado en Arduino diseñado para regular el voltaje de aparatos electrónicos sensibles. El sistema permitía configurar el voltaje exacto que cada dispositivo requería y, en caso de una variación en la corriente, bloqueaba automáticamente el exceso para evitar daños. Esta solución estaba orientada a equipos que requieren un suministro eléctrico preciso para su funcionamiento. El proyecto se complementó con una aplicación móvil en Android Studio, desarrollada en Java, la cual se conectaba a través de Firebase para gestionar y monitorear las configuraciones de voltaje de manera remota.

Proyecto 4 – Plataforma de Venta y Donación Agrícola (2025)

Creación de una plataforma web y móvil orientada a la venta y donación de productos agrícolas. El objetivo principal era brindar a ranchos y granjas un espacio para comercializar sus productos cuando eran rechazados por supermercados, evitando así el desperdicio de alimentos. Además, la plataforma permitía destinar parte de las cosechas a donaciones para personas en situación de necesidad.

La aplicación mostraba información detallada de cada producto, como fecha de siembra, uso de pesticidas, si era orgánico, entre otros datos relevantes para el consumidor. También integraba un mapa con los diferentes puntos de apoyo para facilitar la distribución. El desarrollo web se realizó con HTML, CSS y JavaScript, empleando Firebase como base de datos, mientras que la versión móvil fue desarrollada con Flutter y también conectada a Firebase.

Proyecto 5 – Página Informativa Empresarial (2025)

Creación de una página web informativa para una empresa, enfocada en mostrar sus servicios y datos de contacto. Fue desarrollada con HTML, CSS y JavaScript, alojada en un host privado proporcionado por la empresa.
